  # Termoweb Integration for Home Assistant

  Integration for Termoweb climate control systems.

  ## Installation via HACS

  1. [Install HACS if you haven't already](https://www.hacs.xyz/docs/use/download/download/)
  2. Add this repository as a custom repository in HACS:
     - Go to HACS → Integrations → ⋮ (top right) → Custom repositories
     - Repository: `https://github.com/DanFaudemer/hacs-termoweb`
     - Category: Integration
  3. Click "Install"
  4. Restart Home Assistant
  5. Add the integration via Settings → Devices & Services → Add Integration → Termoweb

  ## Configuration

  Configuration is done via the UI.
